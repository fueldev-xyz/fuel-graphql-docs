# 查询

## `balance`

返回指定地址的特定资产 ID 的 [`Balance!`](/docs/reference/objects/#balance)。

**参数:**

`owner`: [`Address!`](/docs/reference/scalars/#address)

所有者地址。

`assetId`: [`AssetId!`](/docs/reference/scalars/#assetid)

资产 ID。

## `balances`

返回给定地址拥有的每个资产的余额数组的 [`BalanceConnection!`](/docs/reference/objects/#balance)。

**参数:**

`filter`: [`BalanceFilterInput!`](/docs/reference/objects/#balancefilterinput)

指定钱包所有者地址的过滤器。

## `block`

返回有关某个 [`Block`](/docs/reference/objects/#block) 的信息。接受块 ID 或块高度作为参数。

**参数:**

`id`: [`BlockId`](/docs/reference/scalars/#blockid)

块 ID。

`height`: [`U64`](/docs/reference/scalars/#u64)

块高度。

## `blocks`

返回所有生成的块的数组的 [`BlockConnection!`](/docs/reference/objects/#block)。

## `chain`

返回有关用于 API 的目标 Fuel 网络的 [`ChainInfo!`](/docs/reference/objects/#chaininfo)。

## `coin`

返回有关特定 [`Coin`](/docs/reference/objects/#coin) 的详细信息。

**参数:**

`utxoId`: [`UtxoId!`](/docs/reference/scalars/#utxoid)

UTXO 的唯一 32 字节标识符。

## `coins`

返回基于给定所有者和/或资产 ID 的硬币数组的 [`CoinConnection!`](/docs/reference/objects/#coin)。

**参数:**

`filter`: [`CoinFilterInput!`](/docs/reference/objects/#coinfilterinput)

包含所有者地址和可选资产 ID 的过滤器。

## `coinsToSpend`

返回每个资产的可支配 [`[[CoinType!]!]!`](/docs/reference/unions/#cointype) 数组。

**参数:**

`owner`: [`Address`](/docs/reference/scalars/#address)

硬币的所有者地址。

`queryPerAsset`: [`[SpendQueryElementInput!]!`](/docs/reference/objects/#spendqueryelementinput)

请求的资产资源列表。不允许同一个资产 ID 有多个条目。

`excludedIds`: [`ExcludeInput`](/docs/reference/objects/#excludeinput)

要排除的资源。

## `contract`

返回给定合约 ID 的 [`Contract`](/docs/reference/objects/#contract) 信息。

**参数:**

`id`: [`ContractId!`](/docs/reference/scalars/#contractid)

请求合约的合约 ID。

## `contractBalance`

返回给定合约和资产 ID 的 [`ContractBalance!`](/docs/reference/objects/#contractbalance)。

**参数:**

`contract`: [`ContractId!`](/docs/reference/scalars/#contractid)

拥有余额的合约。

`asset`: [`AssetId!`](/docs/reference/scalars/#assetid)

资产 ID。

## `contractBalances`

返回给定合约拥有的所有资产的余额数组的 [`ContractBalanceConnection!`](/docs/reference/objects/#contractbalance)。

**参数:**

`filter`: [`ContractBalanceFilterInput!`](/docs/reference/objects/#contractbalancefilterinput)

合约余额的过滤器。

## `estimateGasPrice`

估计在给定区块范围内的最贵的 gas 价格，并返回 [`EstimateGasPrice!`](/docs/reference/objects/#estimategasprice)。

**参数:**

`blockHorizon`: [`U32!`](/docs/reference/scalars/#u32)

区块范围定义了你在未来估算的区块数。

## `estimatePredicates`

估计断言 gas 并返回 [`Transaction!`](/docs/reference/objects/#transaction)。

**参数:**

`tx`: [`HexString!`](/docs/reference/scalars/#hexstring)

交易的十六进制字符串。

## `health`

如果 API 正在运行则返回 `true`，如果 API 停止则返回 `false`。

## `latestGasPrice`

返回最新区块的 [`LatestGasPrice!`](/docs/reference/objects/#latestgasprice)。

## `message`

返回给定消息 nonce 的 [`Message`](/docs/reference/objects/#message)。

**参数:**

`nonce`: [`Nonce!`](/docs/reference/scalars/#nonce)

消息 nonce。

## `messageProof`

返回给定消息 ID 或交易的 [`MessageProof`](/docs/reference/objects/#messageproof)。

**参数:**

`transactionId`: [`TransactionId!`](/docs/reference/scalars/#transactionid)

消息的交易 ID。

`nonce`: [`Nonce`](/docs/reference/scalars/#nonce)

消息 nonce。

`commitBlockId`: [`BlockId`](/docs/reference/scalars/#blockid)

块 ID。

`commitBlockHeight`: [`U32`](/docs/reference/scalars/#u32)

块高度。

## `messages`

返回给定所有者的消息数组的 [`MessageConnection!`](/docs/reference/objects/#message)。

**参数:**

`owner`: [`Address`](/docs/reference/scalars/#address)

消息的所有者地址。

## `messageStatus`

返回给定消息 [`Nonce`](/docs/reference/scalars/#nonce) 的 [`MessageStatus`](/docs/reference/objects/#messagestatus)。

**参数:**

`nonce`: [`Nonce`](/docs/reference/scalars/#nonce)

消息的 nonce。

## `nodeInfo`

返回有关当前节点的 [`NodeInfo!`](/docs/reference/objects/#nodeinfo)。

## `relayedTransactionStatus`

返回给定中继交易 ID 的 [`RelayedTransactionStatus`](/docs/reference/unions/#relayedtransactionstatus) 详细信息。

**参数:**

`id`: [`RelayedTransactionId!`](/docs/reference/scalars/#relayedtransactionid)

中继交易的 ID。

## `transaction`

返回给定交易 ID 的 [`Transaction`](/docs/reference/objects/#transaction) 详细信息。

**参数:**

`id`: [`TransactionId!`](/docs/reference/scalars/#transactionid)

交易的 ID。

## `transactions`

返回所有交易数组的 [`TransactionConnection!`](/docs/reference/objects/#transaction)。

## `transactionsByOwner`

返回给定地址的所有交易数组的 [`TransactionConnection!`](/docs/reference/objects/#transaction)。

**参数:**

`owner`: [`Address!`](/docs/reference/scalars/#address)

交易的所有者地址。
