# 概述

## Fuel GraphQL API 简介

Fuel GraphQL API 允许您查询 Fuel 区块链上的各种链上数据。它可用于查询交易、余额、区块信息等。您还可以使用它来模拟和提交 Fuel 网络上的交易。

## Playground

Playground 是一个交互式的图形化 IDE，包括查询、突变和类型的参考。它还提供了对底层 GraphQL 模式的查询验证和上下文。

您可以在此处测试 Fuel GraphQL API playground：

[https://testnet.fuel.network/v1/playground](https://testnet.fuel.network/v1/playground)
